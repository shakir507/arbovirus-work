For finding memry issues in the code if segmentation or malloc related error:

To use Valgrind to find memory issues in your C++ code, you can follow these steps:
Install valgring
	sudo apt-get install valgrind
Compile your code with debugging symbols: In order to get more meaningful results from Valgrind, it's recommended to compile your code with debugging symbols enabled. For example, using the -g flag with g++:
	g++ -g your_code.cpp -o your_program--->This means we include -g in the makefile compilation line

Run your program with Valgrind: Use the valgrind command followed by the name of your program to execute it under Valgrind's supervision. For example:

valgrind ./your_program
